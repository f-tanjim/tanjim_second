    TANogle is the front-end design for Google Search, Google Image Search and Google Advanced Search. In this project I have designed three pages and designed them similar to Google pages. On the Google Search page there are two options to search and they are "Google Search" and "I'm Feeling Lucky". "Google Search" is a normal search, but in the "I'm Feeling Lucky" search it takes users directly to the first Google search result for the query, bypassing the normal results page. On the Image Search Page, the user has to type in a query, click the search button (magnifying-glass) and be taken to the Google Image search results for that page. On the Google Advanced Search page, the user is able to provide input for the following four fields: “all these words”,“this exact word or phrase”,“any of these words”,“none of these words”. In my design, I have tried to design pages similar to Google Pages by using only HTML and CSS.
